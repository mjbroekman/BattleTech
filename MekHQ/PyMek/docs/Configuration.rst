=============
Configuration
=============

- Fill out a campaign.config file.
- Any missing values will be substituted with defaults as follows:
 - savedir = /home
 - name = My Campaign
 - compressed = yes
 - same_day = yes
 - validate = no
 - individual = yes
 - lance = yes
 - company = yes
 - scenario = yes
 - mission = yes
 - campaign = yes
